# Milestone Entry Template

Add to `.planning/MILESTONES.md` when shipping a version.

```markdown
## v[X.Y] [Name] (Shipped: YYYY-MM-DD)

**Delivered:** [One sentence describing what shipped]

**Phases completed:** [X-Y] ([Z] plans total)

**Key accomplishments:**
- [Major achievement 1]
- [Major achievement 2]
- [Major achievement 3]

**Stats:**
- [X] files created/modified
- [Y] lines of code
- [Z] phases, [N] plans, [M] tasks
- [D] days from start to ship

**What's next:** [Brief description of next milestone, or "Project complete"]

---
```

If MILESTONES.md doesn't exist, create with header:

```markdown
# Project Milestones: [Project Name]

[Entries newest first]
```
